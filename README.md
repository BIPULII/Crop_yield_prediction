#  AI Assistant Web App

An interactive AI-powered web application built with **Node.js** that allows users to generate intelligent responses using a large language model API.  
This project focuses on a clean UI, fast responses, and a simple developer-friendly setup for running and deploying the app.

---

##  Features

- 💬 Real-time AI chat interface
- ⚡ Fast response handling
- 🔐 Secure API key management using environment variables
- 🧩 Modular and maintainable project structure
- 🌐 Ready for deployment on modern hosting platforms

---

##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js
- **AI Integration:** Gemini API
- **Environment Management:** dotenv
